# README

# 開発言語
* Ruby 3.0.1
* Ruby on Rails 6.0.4

# 就業Teamの技術
* devise
* Ajaxコメント機能
* お気に入り機能

# カリキュラム以外の技術
* ransack
# 実行手順
```
git clone git@github.com:chihiro-kakizaki/graduation_app.git
cd WD_manage
bundle install
rails db:create && rails db:migrate
rails s
```
